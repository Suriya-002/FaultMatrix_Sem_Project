## Project Overview

**FaultMatrix** analyzes electronic device repair guides to identify failure patterns using:
- Natural Language Processing (GLiNER, spaCy)
- Topic Modeling (BERTopic)
- Large Language Models (AWS Bedrock + Claude)
- Interactive 3D Visualization (Three.js)

For technical details, see `/docs/architecture/ARCHITECTURE.md`
